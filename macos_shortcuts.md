# MacOS Shortcuts

| Key Combinations | Action |
|------------------|--------|
| `Command (⌘)`+ `K` | In terminal: Clear screen and scrollback buffer |
| `Shift (⇧)` + `Command (⌘)`+ `3` | Screenshot: Entire screen |
| `Shift (⇧)` + `Command (⌘)`+ `4` | Screenshot: Selected area |
| `Shift (⇧)` + `Command (⌘)`+ `5` | Screenshot toolbar |
| `Command (⌘)`+ `Q` | Quit app |
